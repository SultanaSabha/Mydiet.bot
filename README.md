MyDiet.Bot is an AI-powered conversational nutrition assistant built using Python and Streamlit that provides personalized diet and food-related guidance through natural language interaction. The application leverages retrieval-augmented generation (RAG) to answer user queries by combining a language model with a vector database for accurate, context-aware responses.

The system ingests nutrition and diet-related data, generates embeddings, and stores them in a ChromaDB vector store. When a user asks a question, the application retrieves the most relevant information using semantic search and generates a concise, user-friendly response. This approach improves reliability compared to standalone language models.

The project demonstrates practical experience in LLM-based applications, time-efficient data ingestion pipelines, and interactive web app development. It highlights real-world skills in building AI assistants, managing embeddings, and deploying user-facing analytics tools.

Tech Stack:
Python, Streamlit, LangChain, ChromaDB, Vector Embeddings, LLMs

Key Features:
Conversational AI interface for diet and nutrition queries
Retrieval-Augmented Generation (RAG) for accurate responses
Efficient document ingestion and embedding pipeline
Interactive web-based UI using Streamlit
Modular, scalable project structure
